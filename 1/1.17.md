```c++
#include <iostream>

int main()
{
    int currVal = 0, val = 0;
    if (std::cin >> currVal)
    {
        int cnt = 1;
        while (std::cin >> val)
        {
            if (val == currVal)
            {
                ++cnt;
            } else {
                std::cout << currVal << " occurs "
                          << cnt << " times" << std::endl;
                currVal = val;
                cnt = 1;
            }
        }
        std::cout << currVal << " occurs "
                  << cnt << " times" << std::endl;
    }
    return 0;
}
```

如果输入的值都是相等的，则输出的结果是这个值出现的次数，并且只有一行，即:

```txt
VALUE occurs N times
```

如果输入的值都是不一样的，则每个值都输出一遍，即:

```txt
VALUE1 occurs 1 times
VALUE2 occurs 2 times
VALUE3 occurs 3 times
...
```
